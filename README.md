# 🧞‍♂️ Food Genie - Frontend

Food Genie is a modern, responsive web application designed to streamline food discovery, ordering, and dynamic restaurant browsing. This repository contains the frontend client interface, engineered with speed, scalability, and clean UI design principles.

---

## 🚀 Tech Stack & Setup

* **Framework:** React 19 (Latest stable features)
* **Build Tool:** Vite 8 (Optimized for ultra-fast development and Hot Module Replacement)
* **State Management:** Redux Toolkit (`@reduxjs/toolkit` & `react-redux`)
* **Routing:** React Router v7 (`react-router-dom`)
* **API Client:** Axios
* **Notifications:** React Toastify
* **Ecosystem:** Node.js & MERN Stack architecture

---

## ✨ Features

* **Intuitive UI/UX:** Clean, fully responsive interfaces across mobile, tablet, and desktop views.
* **Global State Management:** Structured Redux store handling asynchronous logic and dynamic data flow seamlessly.
* **Proxy Configuration:** Configured development server proxy to cleanly handle API routing to local backend instances.
* **Environment Safety:** Strict separation of environment variables (`.env`) to ensure secure third-party credentials.

---

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see 
[this documentation](https://react.dev/learn/react-compiler/installation).
